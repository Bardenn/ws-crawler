# Lessons Overview

These short lessons break the search tool into explainable components for the
coursework video. Each lesson is written so it can be used as a teaching script:
what the component does, how the current implementation works, where GenAI was
useful, where human checking mattered, and the trade-offs between alternative
approaches.

Suggested video use:

- Use lessons 1-4 for the technical walkthrough.
- Use lesson 5 for demonstrating testing and evidence.
- Use lesson 6 for the required critical GenAI reflection.
